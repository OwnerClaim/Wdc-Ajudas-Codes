Atalhos

Atalhos do Computador

  • Ctrl + C: Copiar
  • Ctrl + V: Colar
  • Ctrl + X: Cortar (Você remove o item de um local e cola em outro)
  • Ctrl + y: Volta itens removidos pelo Ctrl + Z
  • Alt + Tab: Alternar entre janelas
  • Ctrl + Shift + Esc: Abrir o Gerenciador de Tarefas
  • Windows + D: Mostrar área de trabalho

Atalhos do Excel

  • Shift + Scroll (Bolinha do mouse): Move a planilha para esquerda ou direita
  • Ctrl + S: Salvar
  • F4: Repetir última ação
  • Ctrl + Shift + L: Aplicar/Remover filtros
  • Alt + =: AutoSoma
